6非機能要件

🔒 セキュリティ

  • HTTPS通信必須（SSL/TLS）必須
  • パスワードはbcrypt等でハッシュ化保存必須
  • セッション管理（タイムアウト設定）
  • CSRF対策
  • ロールベースアクセス制御（RBAC）

⚡ パフォーマンス

対象目標値
出勤簿一覧 表示< 2秒
打刻記録 レスポンス< 1秒
月次集計データキャッシュ活用

🛡️ 可用性

  • 稼働率 99.9% 以上（月間ダウンタイム < 44分）
  • データバックアップ: 日次

🌐 多言語対応

  • i18n対応: 日本語・英語を最低限サポート
  • UIテキストはロケールファイルで管理

📱 レスポンシブ対応

  • PC / タブレット / スマートフォン対応
  • モバイルからの打刻操作を優先考慮
